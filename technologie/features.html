Einfach & Intuitiv

Sykus 3 ist von der Oberfläche bis hin zur Bedienung der einzelnen Funktionen einfach und selbsterklärend aufgebaut. Somit können Schüler und Lehrer das System intuitiv und ohne Anleitung benutzen.

Internetfilter

Der Internetfilter der neusten Generation umfasst die Blacklists von zwei Anbietern und erfasst ein enormes Spektrum an Webseiten. Durch die Vielzahl der Kategorien und Einstellungen kann der Filter exakt auf die jeweiligen Bedürfnisse angepasst werden.

Alle Filterkategorien können entweder für alle Benutzer oder nur für Schüler gesperrt werden. So können Lehrer einen weniger stark gefilterten Internetzugang nutzen. Außerdem können manuelle White- und Blacklist-Einträge angelegt werden. Somit bietet der Internetfilter alle Möglichkeiten, um den Internetzugriff ohne unnötige Einschränkungen optimal abzusichern.

Webinterface

Alle Funktionen sind in einem übersichtlichen Webinterface zusammengefasst. So können Sie nicht nur von jedem beliebigen Gerät aus, sondern auch weltweit über das Internet auf das System zugreifen.

Das Webinterface wird über eine eigene Toplevel-Domain erreicht, die frei ausgewählt werden kann. Der Zugriff erfolgt ausschließlich über eine sichere SSL-Verbindung.

Firewall & Sicherheit

Durch die integrierte Firewall ist das Netzwerk gegen Angriffe von außen geschützt. Alle Daten liegen sicher auf dem Server innerhalb der Schule. Der Zugriff von außen ist nur über eine verschlüsselte Verbindung möglich.

Benutzerimport

Am Anfang des Schuljahres werden die Benutzerkonten für Schüler und Lehrer mithilfe einer CSV-Liste importiert. Dabei werden neue Benutzer hinzugefügt, Schüler automatisch in die neue Klasse versetzt und alte Benutzer gelöscht. Der Schuljahreswechsel mit Sykus 3 dauert somit nur wenige Sekunden.

Sichere Erstanmeldung

Sobald ein neuer Benutzer erstellt wird, bekommt dieser ein zufälliges Passwort zugewiesen. Die Initialpasswörter werden dann automatisch in einer Druckvorlage zusammengefasst und können dann nach Klassen sortiert ausgedruckt werden. So erhält jeder Benutzer sein individuelles Initialpasswort sicher in Papierform.

Die Ausdrucke können dann von den Klassenlehrern individuell an jeden Schüler verteilt werden, sodass maximale Sicherheit gewährleistet ist. Die Erstanmeldung muss nur ein einziges Mal per Benutzer erfolgen, beim Schuljahreswechsel bleibt das persönliche Passwort erhalten.

Benutzerverwaltung

Die Benutzerverwaltung bietet eine Übersicht über alle im System vorhanden Schüler, Lehrer und weiteren Benutzer. Neue Benutzer können manuell angelegt werden oder über die Importfunktion hinzugefügt werden.

Gruppenverwaltung

Organisieren Sie Ihre Schüler schnell und intuitiv in logische Gruppen. Sie können Gruppen für z.B. Kurse, Klassenfahren oder Projektgruppen erstellen und so gezielt Informationen mit der Dateiablage, dem Nachrichtensystem und dem Kalender austauschen.

Softwareverteilung

Mit der Softwareverteilung, die auf dem Imaging-Prinzip beruht, können Sie mit einem Knopfdruck alle Rechner vollautomatisch mit dem neusten Image bespielen. Dieses Image enthält das Betriebssystem und die individuell ausgewählten Softwarepakete.

Automatisch am Wochenende oder manuell durch den Administrator ausgelöst wird nachts ein neues Image in einer Virtuellen Maschine auf dem Server erstellt, welches dann vollautomatisch auf die Clients aufgespielt wird. Dabei werden die Rechner per Wake-on-LAN und über PXE-Boot in das Installationsystem gestartet, das System wird installiert und die Rechner danach wieder heruntergefahren.

Softwarepakete

Mit der Softwarepaket-Auswahl kann der Administrator gezielt die Software auswählen, die im Schulbetrieb eingesetzt werden soll. Wenn eine Software nicht in der umfangreichen Liste vorhanden ist, kann diese auf Anfrage kostenlos hinzugefügt werden.

Rechnerverwaltung

In einer übersichtlichen Liste sind alle Rechner aufgelistet, die mit dem Client-Image bespielt werden. Hier sieht der Administrator den aktuellen Status des Rechners sowie weitere Hardwareinformationen.

Raumsteuerung

Mit der Raumsteuerung behalten Sie alle Schülerbildschirme im Computerraum im Blick. Mit nur einem Klick können Sie Bildschirme, Internet, Ton oder Drucker sperren und freigeben.

Dateiablage

Jeder Benutzer erhält ein persönliches Verzeichnis, in dem Daten abgelegt werden können. Es gibt außerdem individuelle Dateifreigaben für Benutzergruppen, in denen die Gruppenmitglieder Dateien austauschen können. Alle Dateiablagen sind weltweit über eine gesicherte Verbindung erreichbar.

Durch die automatische Speicherplatzverwaltung erhält jeder Benutzer ausreichend Speicherkapazität, ohne dass die Festplattenauslastung des Servers einen bestimmten Wert überschreitet. Somit ist ein optimales Verhältnis zwischen Flexibilität und Begrenzung sichergestellt.

Kalender

Mit dem Kalendersystem können Sie Ihre persönlichen Termine organisieren und Termine für Ihre Gruppen, Klassen und Jahrgänge eintragen. Der Kalender umfasst auch einen Klausurplan und eine Resourcenverwaltung bzw. Raumbelegungsplan.

Nachrichten

Das Nachrichtensystem ermöglicht die Kommunikation zwischen den Benutzern. Schüler können Nachrichten an Lehrer und eigene Gruppen schreiben. Lehrer können mit einzelnen Schülern oder Lehrern, allen Lehrern, Gruppen, Klassen sowie Jahrgängen kommunizieren.

WLAN

Mit dem integrierten RADIUS-Server können sich Schüler und Lehrer mit ihren bereits bekannten Anmeldedaten am schuleigenen WLAN-Netzwerk anmelden. Das mühsame Austauschen von WLAN-Keys entfällt somit.

Grundschulen

Für den Einsatz an Grundschulen wurden die Funktionen so angepasst, dass auch jüngere Schüler problemlos das System benutzen können. Außerdem wurden Lernprogramme integriert, die speziell für Grundschüler entwickelt wurden. Durch die geringen monatlichen Gebühren und den Wegfall von Lizenzkosten ist Sykus 3 auch für kleine Grundschulen geeignet.

Drucker

Netzwerkfähige Drucker bzw. Printserver können mit wenigen Klicks ins System integriert werden. Der Druckerzugriff kann für verschiendene Räume oder Rechnergruppen getrennt freigeschaltet werden. Alle Sykus-Clients können dann auf die jeweils erlaubten Drucker automatisch zugreifen.

Logs

Mit ausführlichen Logs für Benutzeranmeldung, Webinterface-Zugriff und Administrator-Aktionen können alle Vorgänge nachvollzogen werden. Die Logs sind nicht veränderbar und von allen Administratoren einsehbar.

SMART Boards

Interaktive Tafelsysteme von SMART Technologies können problemlos mit Sykus 3 benutzt werden. Die nötige Software kann mit einem Klick vom Administrator installiert werden und integriert sich nahtlos in das System.
