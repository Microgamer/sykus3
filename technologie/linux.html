Stabil durch Linux

Durch den Einsatz von Linux auf dem Server und den Clients ist ein homogenes System gewährleistet, welches maximale Stabilität bietet. So können stets aktuelle Softwareversionen benutzt und verschiedene Hardware parallel betrieben werden.

Als Basis für die Client-Rechner benutzen wir das Ubuntu-Derivat Xubuntu, welches alle sechs Monate auf die neuste Version aktualisiert wird. Zusätzlich zur großen Open-Source-Community wird Ubuntu von der Softwarefirma Canonical entwickelt.

Keine Softwarelizenzen

Da Sykus 3 ausschließlich Open-Source-Software benutzt, fallen weder für das Betriebssystem noch für Anwendungsprogramme Lizenkosten an. Durch die Integration von Webdiensten der großen Schulbuchverlage können alle Ansprüche im Schuleinsatz abgedeckt werden.

Softwarepakete

Mit der Softwarepaket-Auswahl kann der Administrator gezielt die Software auswählen, die im Schulbetrieb eingesetzt werden soll. Wenn eine Software nicht in der umfangreichen Liste vorhanden ist, kann diese auf Anfrage kostenlos hinzugefügt werden.

Softwareverteilung

Mit der Softwareverteilung, die auf dem Imaging-Prinzip beruht, können Sie mit einem Knopfdruck alle Rechner vollautomatisch mit dem neusten Image bespielen. Dieses Image enthält das Betriebssystem und die individuell ausgewählten Softwarepakete.

Automatisch am Wochenende oder manuell durch den Administrator ausgelöst wird nachts ein neues Image in einer Virtuellen Maschine auf dem Server erstellt, welches dann vollautomatisch auf die Clients aufgespielt wird. Dabei werden die Rechner per Wake-on-LAN und über PXE-Boot in das Installationsystem gestartet, das System wird installiert und die Rechner danach wieder heruntergefahren.
